# PWA Projects

Collection of Progressive Web Apps.

## Projects

- **fuel-manager** — Gestione Benzina: track rides, fuel consumption, shared expenses and reimbursements.

## Deploy

Tutte le PWA seguono la stessa convenzione (un `docker-compose.yml` per app, porta
solo su loopback, reverse proxy sul NAS via Tailscale). Vedi [`DEPLOY.md`](DEPLOY.md)
per la guida generale e il `DEPLOY-NAS.md` dentro ogni app per le note specifiche.
